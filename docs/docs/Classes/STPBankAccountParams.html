STPBankAccountParams

@interface STPBankAccountParams : NSObject <STPFormEncodable>

Representation of a user’s bank account details. You can assemble these with information that your user enters and then create Stripe tokens with them using an STPAPIClient. - see: https://stripe.com/docs/api#create_bank_account_token
