STPCardParams

@interface STPCardParams : NSObject <STPFormEncodable>

Representation of a user’s credit card details. You can assemble these with information that your user enters and then create Stripe tokens with them using an STPAPIClient. - see: https://stripe.com/docs/api#cards

  • currency
